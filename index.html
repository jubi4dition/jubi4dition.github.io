Welcome!

Hello and welcome to my GitHub site. I started developing web application with PHP in 2011, but learned programming with Java some years before. The PHP-Framework Codeigniter was the first one I used and that was the one, which i used to programm my first web application YourContacts. I don't really use or look at something about Codeigniter anymore, but from time to time I still try to update my project YourContacts. So why i don't use it anymore? Because in the middle of 2012 i saw the PHP-Framework Laravel. With Laravel 3, I developed my web applications LaravelContacts, JubiTripBooker and Juballery. The frontend developing is mostly done with jQuery and the CSS-Framework Bootstrap. The next step will be to switch to Laravel 4.
